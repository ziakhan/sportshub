# 📣 Club & league-wide announcements

> **✅ SHIPPED 2026-07-09 (Phases 1+3)** — club & league Messages composers with audiences,
> drafts, send-all, MessageLog audit, and CASL consent enforced at send time
> ([[season-continuity-plan]] §4–5, gap-audit P0 #9). This doc remains as the original
> requirement statement.

**Tier 2 · effort M · from layer1.** Communication is entirely per-team; there's no club-wide or league-wide broadcast, and multi-team event push is API-only.

## Problem
Only team-scoped chat/announce; league-wide event push has no UI.

## Scope
- Club-wide announcement to all its families
- League-wide announcement/event push to all teams (UI over the existing multi-team API)
- Deduped bell + email fan-out

## Acceptance
- A club/league operator can message all their families in one action

## Dependencies
none
