# 👪 Co-guardian / two-household support

**Tier 2 · effort M · from layer2.** Player.parentId is a single FK, so divorced/two-household families (the norm) can't both see the child, get comms, or share payments.

## Problem
One parent per player; second guardian/grandparent can't be reached.

## Scope
- Co-guardian model (multiple guardians per player, roles/permissions)
- Both households see the child + receive comms
- Optional split payments across households

## Acceptance
- A second guardian can be added to a child and receives schedule + comms

## Dependencies
player-safety-medical
