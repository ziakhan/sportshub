# 🎟️ Financial aid + sibling/promo discounts + tax receipts

**Tier 2 · effort L · from layer2.** Accessibility is central to Ontario youth sport (Jumpstart/KidSport) and multi-kid discounts are near-universal; only a manual 'waive' exists. Plus Canadian year-end tax receipts.

## Problem
No aid intake, no coupon/discount model, no sibling logic, no year-end receipts.

## Scope
- Financial aid application + voucher-code/partial-subsidy tracking (Jumpstart/KidSport)
- Promo/discount codes + sibling/family multi-child discounts
- Canadian year-end tax/childcare receipts (org, amount, child)

## Acceptance
- A family can apply a subsidy/discount code; siblings get the family rate
- Parents can download a year-end receipt

## Dependencies
none
