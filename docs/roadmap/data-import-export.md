# 📦 Data import (migration) + export

**Tier 2 · effort L · from layer2.** Pure GTM blocker — you can't displace RAMP/TeamSnap/Exposure without importing a club's existing rosters + schedules; export is needed for PIPEDA portability + anti-lock-in.

## Problem
Only order-sheet CSV export; no roster/schedule import tooling.

## Scope
- CSV/roster import (players, teams, schedules) with mapping + validation
- Full data export per club/family
- Migration guides for the top incumbents

## Acceptance
- A club can import an existing roster + schedule
- A club/family can export their data

## Dependencies
none
