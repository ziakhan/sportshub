# 💸 Pay referees via Connect

**Tier 1 · effort M · from layer2.** A per-game fee is captured and displayed but referees are never paid through the platform — no obligation/payout/Connect rail (league fees are fully wired; this mirrors them).

## Problem
Referee fee is display-only; booking/finalize create no payment.

## Scope
- Referee Stripe Connect onboarding
- Create a payout obligation on game assignment/finalize
- Who-owes-whom + payout status view for the league

## Acceptance
- A league can pay an assigned referee through the platform and see payout status

## Dependencies
none
